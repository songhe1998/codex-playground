# AGENTS.md

This repository is a sandbox for exploring Codex website generation and Codex Cloud behavior.

## Scope

- Keep changes inside this repository.
- Prefer small, easy-to-review experiments.
- Do not introduce unnecessary dependencies.

## Default workflow

- If the task is to create or revise a website experiment, keep it minimal.
- Prefer plain HTML, CSS, and JavaScript unless a framework is explicitly requested.
- If a framework is needed, prefer Vite for small experiments.

## Validation

- Run `npm run build` before finishing when the task changes website code.
- Report whether the build passed.

## Notes

- The purpose of this repo is exploration, not production polish.
- Favor fast iteration and clear diffs.
